<task type="auto">
  <name>Task 1: Build wheel and validate in clean venv</name>
  <files></files>
  <read_first>pyproject.toml</read_first>
  <action>
Build a wheel and validate it works in a clean environment:

1. Build the wheel:
   ```bash
   python -m build
   ```
   (If `build` is not installed, install it first: `pip install build`)

2. Create a clean temporary venv:
   ```bash
   python -m venv /tmp/ream-xlsx-test-venv
   ```

3. Install the built wheel (NOT editable) plus pytest:
   ```bash
   /tmp/ream-xlsx-test-venv/bin/pip install dist/ream_xlsx-0.1.0-py3-none-any.whl
   /tmp/ream-xlsx-test-venv/bin/pip install pytest
   ```

4. Run the full test suite using the clean venv's Python:
   ```bash
   /tmp/ream-xlsx-test-venv/bin/pytest tests/ -v
   ```

5. Verify the CLI works from the clean venv:
   ```bash
   /tmp/ream-xlsx-test-venv/bin/ream-xlsx --version
   ```

6. Verify README is in the sdist:
   ```bash
   tar tzf dist/ream_xlsx-0.1.0.tar.gz | grep README
   ```

7. Clean up:
   ```bash
   rm -rf /tmp/ream-xlsx-test-venv
   ```

If any step fails, diagnose and report the issue. Do NOT modify package files -- this is a validation-only task.
  </action>
